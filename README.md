# Simple Flight Simulator

A basic 3D flight simulator written in C. This project was created to practice C programming concepts, 3D coordinate systems, and basic game loops.

## Technology Stack
This project uses **Raylib**, a simple and easy-to-use library to enjoy videogames programming.

## Prerequisites
To compile and run this project, you need to install Raylib on your system.
* You can download the Windows Installer from the official GitHub releases: [Download Raylib](https://github.com/raysan5/raylib/releases)

## Controls
* **W / S:** Move Forward / Backward
* **A / D:** Move Left / Right
* **SPACE:** Move Up
* **SHIFT:** Move Down
